# helper for printing subject summary with tree structure showing best status across runs

helper for printing subject summary with tree structure showing best
status across runs

## Usage

``` r
print_subject_summary_tree(subject_jobs_df, subject_id)
```

## Arguments

- subject_jobs_df:

  A data.frame of jobs for a specific subject across all sequence IDs

- subject_id:

  The subject ID being summarized
